# `.reinguard/policy`

Normative documents for this repository: invariants, review and merge
discipline, exception policy, and related **must-follow** rules.

This directory is **not** indexed by `rgd knowledge index`. The Adapter
layer (`.cursor/`, `AGENTS.md`) references policy files by path. For
judgment aids and background knowledge, use `.reinguard/knowledge/` instead
(ADR-0010, ADR-0011).

## Placement

- **Policy** — text agents must comply with (dispositions, merge gates,
  traceability).
- **Knowledge** — improves decisions when read; not mandatory policy text.

A future `catalog.yaml` may list policy ids and paths for tooling; that is
out of scope until needed.
